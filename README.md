## 📋 Course Overview

This repository contains coursework, projects, and code examples for **Java Programming**. The course introduces the concepts of programming using Java 10+, covering the basics of Java language syntax alongside elementary data structures, input/output statements, selection, iteration, methods, and arrays.


**Prerequisite:** An Introduction to Programming or equivalent experience.

---

## 🎯 What You'll Learn

| Area | Topics Covered |
|---|---|
| Java Basics | Syntax, data types, input/output statements |
| Selection | If, If/Else, Switch statements |
| Mathematics | Mathematical functions and operations |
| Strings | Concatenation, length, equals and compare |
| Loops | While, For, Foreach, Do/While |
| Methods | Defining, invoking, and passing arguments |
| Arrays | Single-dimensional and multidimensional |
| OOP | Objects, classes, constructors, static contents |
| Inheritance | Super/sub classes, overriding methods |
| Exception Handling | Exception types, throwing and catching, finally |

---

## 📌 Course Objectives

Successfully completing this course, students will be able to:

1. Write syntactically correct Java programs using core language features
2. Apply selection and iteration structures to control program flow
3. Define and invoke methods with proper use of arguments and return types
4. Manipulate strings using built-in Java string methods
5. Create and traverse single-dimensional and multidimensional arrays
6. Design programs using Object-Oriented principles including classes, objects, and constructors
7. Implement inheritance and polymorphism in Java class hierarchies
8. Handle runtime errors using Java's exception handling mechanisms
