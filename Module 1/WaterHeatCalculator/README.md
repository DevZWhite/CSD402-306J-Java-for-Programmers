# Water Heat Calculator

A simple Java program that calculates the energy required to heat water based on mass, temperature change, and specific heat capacity.

## Description

This project is a console application that accepts input values for:
- water mass
- initial temperature
- final temperature
- specific heat capacity

The program computes the amount of heat required to raise the water temperature.

## Getting Started

1. Open the project in Visual Studio Code.
2. Build the project using your Java build tools or the VS Code Java extensions.
3. Run the main class from the `src` folder or the generated `bin` output.

## Usage

Provide the required input values when prompted. The program will display the calculated heat energy value.

## Folder Structure

- `src`: Java source files
- `bin`: compiled output files
- `lib`: optional dependencies

## Notes

- Ensure Java is installed and configured in your environment.
- Use the VS Code Java extensions for easier editing, building, and debugging.
